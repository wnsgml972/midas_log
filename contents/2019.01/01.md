### Enum Class

* Enum의 범위를 제한하기 위해 Enum Class를 사용한다.
* FeatureSolid 참조할 것

~~~cpp
class ABC
{
public:
    enum class FileState { WRITE, READ };
};

int main()
{
    ABC::FileState file1 = ABC::FileState::WRITE;
}
~~~
