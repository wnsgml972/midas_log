# 단축기

## Window 10 단축기
* 화면분할기능 __( ```윈도우키 + 방향키 ``` )__
* 가상데스크톱.작업화면보기 __( ```윈도우키 + tap``` )__
* 가상데스크톱.추가 __( ```Ctrl + 윈도우키 + D``` )__
* 가상데스크톱.화면전환  __( ```Ctrl + 윈도우키 + 방향키``` )__
* 알림센터열기 __( ```윈도우키 + A``` )__
* 찾기 __( ```윈도우키 + S``` )__
* 설정창열기 __( ```윈도우키 + I``` )__
