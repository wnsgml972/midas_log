# 단축기

## Windows 10 단축기
* 이모지 __( ```Win +  .``` )__
* 클립보드 __( ```Win +  V``` )__
* 캡처 __( ```Win +  Shift + S``` )__
* 화면분할기능 __( ```Win + 방향키 ``` )__
* 찾기 __( ```Win + S``` )__
* 작업폴더열기 __( ```Win + E``` )__
* 가상데스크톱.작업화면보기 __( ```Win + tap``` )__
* 가상데스크톱.추가 __( ```Ctrl + Win + D``` )__
* 가상데스크톱.화면전환  __( ```Ctrl + Win + 방향키``` )__
* 알림센터열기 __( ```Win + A``` )__
* 설정창열기 __( ```Win + I``` )__
* 작업표시줄아이콘바로실행 __( ```Win + 1~10``` )__
* 바탕화면바로가기 __( ```Win + D``` )__
* 로그오프 __( ```Win + L``` )__
